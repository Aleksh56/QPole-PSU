# QPole-PSU

QPole-PSU is a web application that serves as a tool for creating and taking mini-surveys and quizzes.

## Technologies Used

### Front-end
- React
- JavaScript
- Effector (for state management)
- Material-UI (MUI)
- Vite (as the build tool)
- Hapi (for Server-Side Rendering - SSR)

### Back-end
- Python
- Django

### Database
- MySQL
